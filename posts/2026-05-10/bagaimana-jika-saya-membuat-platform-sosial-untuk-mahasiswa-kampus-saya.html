Bagaimana Jika Saya Membuat Platform Sosial untuk Mahasiswa Kampus Saya?

10 Mei 2026, 14:19

Akhir-akhir ini saya sering memikirkan sesuatu.

Bagaimana jika saya membuat sebuah platform online khusus untuk mahasiswa di kampus saya sendiri?

Bukan cuma website biasa. Tapi sesuatu yang benar-benar dipakai mahasiswa setiap hari.

Mungkin seperti Facebook versi kampus.

Awalnya Hanya Ide Random

Semua ini sebenarnya berawal dari pemikiran random saja.

Saya suka coding. Saya suka membuat project. Dan saya terinspirasi dari kisah Mark Zuckerberg dulu ketika membuat Facebook untuk kampusnya.

Lalu saya mulai berpikir:

“Bagaimana jika saya membuat sesuatu seperti itu di kampus saya?”

Sebuah tempat dimana mahasiswa bisa:

  • Membuat confession anonim
  • Mencari partner tugas
  • Memberi review dosen dan mata kuliah
  • Melihat info kampus terbaru
  • Menjual jasa atau skill mereka
  • Membangun reputasi mereka sendiri

Semua dalam satu platform.

Kenapa Saya Tertarik Dengan Ide Ini?

Karena saya sadar satu hal.

Mahasiswa sebenarnya sangat aktif di internet.

Mereka suka:

  • scroll social media
  • membahas drama kampus
  • mencari info dosen
  • mencari teman kelompok
  • membagikan meme
  • mencari pengakuan sosial

Tapi anehnya, belum ada tempat khusus yang benar-benar dibuat untuk kehidupan mahasiswa kampus saya sendiri.

Fitur yang Saya Bayangkan

1. Confession Anonim

Mahasiswa bisa mengirim confession anonim seperti:

  • curhat kampus
  • crush
  • pengalaman lucu
  • cerita absurd

Mungkin ini bisa jadi fitur paling ramai.

2. Review Dosen

Mahasiswa bisa memberi review dosen dan mata kuliah.

Karena jujur saja, hampir semua mahasiswa pasti penasaran:

  • dosen ini killer atau tidak?
  • banyak tugas atau tidak?
  • susah lulus atau tidak?

3. Cari Partner Tugas

Kadang mahasiswa kesulitan mencari kelompok atau partner project.

Mungkin platform ini bisa membantu mempertemukan mahasiswa berdasarkan skill atau jurusan mereka.

4. Marketplace Skill

Mahasiswa bisa menawarkan jasa:

  • desain
  • coding
  • edit video
  • fotografi
  • dan lain-lain

Bagian yang Paling Menarik

Saya membayangkan platform ini punya sistem reputasi.

Semakin aktif seseorang membantu mahasiswa lain, semakin tinggi reputasinya.

Mungkin akan ada:

  • badge
  • level
  • ranking mahasiswa aktif
  • top contributor

Sedikit seperti game. Sedikit seperti social media.

Tapi Jujur Saja...

Saya sendiri tidak tahu apakah ide ini akan berhasil atau tidak.

Bahkan saya ragu apakah orang-orang akan melihat blog ini.

Karena kebanyakan mahasiswa di sekitar saya mungkin tidak terlalu akrab dengan platform seperti GitHub atau blog pribadi programmer.

Tapi mungkin itu bukan masalah utama.

Karena terkadang project besar memang dimulai dari sesuatu yang kecil dan random.

Mungkin Tidak Akan Berhasil

Mungkin project ini gagal.

Mungkin tidak ada yang memakai.

Mungkin cuma jadi salah satu project random saya.

Tapi entah kenapa, saya tetap ingin mencoba membuatnya.

Karena saya penasaran:

“Bagaimana rasanya membuat sesuatu yang benar-benar hidup dan dipakai banyak orang?”

Penutup

Untuk sekarang ini masih sebatas ide.

Belum ada nama pasti. Belum ada desain pasti. Belum ada roadmap yang jelas.

Tapi mungkin suatu hari nanti, ide random ini benar-benar berubah menjadi sesuatu yang besar.

Dan kalau itu terjadi, saya rasa itu akan menjadi pengalaman yang sangat menarik.
