Bagaimana Jika Saya Membuat “Facebook” Versi Kampus Saya Sendiri?

Sebuah ide random dari mahasiswa yang suka coding.

Beberapa waktu lalu saya kepikiran satu hal random.

Dulu Mark Zuckerberg membuat Facebook awalnya hanya untuk kampusnya sendiri. Bukan langsung untuk seluruh dunia.

Dan saya mulai berpikir...

“Bagaimana kalau kampus saya punya platform sosialnya sendiri?”

Tempat dimana mahasiswa bisa:

  • Confession anonim
  • Mencari partner tugas
  • Memberi review dosen
  • Membagikan info kampus
  • Menjual skill mereka
  • Membangun reputasi digital

Awalnya Hanya Ide Iseng

Jujur saja, awalnya ini bukan ide startup besar atau bisnis miliaran.

Saya hanya suka coding.

Dan saya merasa menarik jika bisa membuat sesuatu yang benar-benar dipakai oleh mahasiswa di kampus saya sendiri.

Bukan sekadar project tugas.

Tapi sesuatu yang hidup.

Fitur yang Saya Bayangkan

1. Confession Anonim

Mahasiswa bisa mengirim confession anonim.

Bisa lucu, random, curhat, atau sekadar cerita kehidupan kampus.

2. Review Dosen & Mata Kuliah

Sebelum mengambil kelas, mahasiswa bisa melihat review dari senior:

  • Apakah dosennya santai?
  • Tugasnya banyak?
  • Susah lulus?

3. Cari Partner Tugas

Kadang mahasiswa kesulitan mencari kelompok.

Jadi platform ini bisa membantu mencari partner berdasarkan:

  • Jurusan
  • Skill
  • Jam aktif
  • Keahlian coding/desain

4. Marketplace Skill Mahasiswa

Mahasiswa bisa menjual skill mereka:

  • Desain
  • Edit video
  • Coding
  • Fotografi
  • Jasa presentasi

5. Reputation System

Saya juga membayangkan sistem level seperti game.

Semakin aktif membantu mahasiswa lain, semakin tinggi reputasinya.

Jadi mahasiswa punya “identitas digital” di kampus.

Masalahnya...

Saya sadar ada satu masalah besar.

Banyak mahasiswa di sekitar saya mungkin belum terlalu terbiasa menggunakan platform seperti GitHub atau website komunitas.

Jadi tantangan terbesar bukan coding.

Tapi:

“Bagaimana membuat mahasiswa mau membuka dan memakai platform ini?”

Karena produk yang bagus bukan hanya soal teknologi.

Tapi soal kebiasaan dan komunitas.

Apakah Ini Akan Benar-Benar Terjadi?

Saya juga tidak tahu.

Bisa saja project ini gagal total.

Bisa saja tidak ada yang memakai.

Tapi menurut saya tetap menarik untuk dicoba.

Karena daripada hanya memikirkan ide, saya ingin mencoba membangunnya.

Penutup

Mungkin ini hanya project kecil.

Mungkin juga suatu hari bisa menjadi sesuatu yang besar.

Tapi yang jelas, saya ingin membuat sesuatu yang benar-benar hidup dan dipakai oleh orang lain.

Dan semuanya dimulai dari satu ide random:

“Bagaimana jika kampus saya punya Facebook-nya sendiri?”
